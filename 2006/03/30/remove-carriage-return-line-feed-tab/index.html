Remove Carriage Return, Line Feed, Tab in TSQL

Mar 30th, 2006

{

– remove carriage return, line feed, and tab from a field in tsql
REPLACE(REPLACE(REPLACE(MyField, CHAR(10), ”), CHAR(13), ”), CHAR(9), ”)

}
